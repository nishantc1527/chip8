# Chip 8
Chip 8 emulator made in Go. Games are taken from https://github.com/badlogic/chip8/tree/master/roms 

# Download
```
go get -u github.com/nishantc1527/chip8
```

Make sure to put $GOPATH/bin in your path.

# Usage
```
chip8 <path-to-rom>
```
